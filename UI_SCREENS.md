```

│   ├─ Score Display

│   └─ Motivational Message
│
├── Habits Tab
│   ├─ Good Habits List
│   ├─ Bad Habits List
│   ├─ + Create Habit Modal
│   └─ Child Habits View
│
├── Todos Tab
│   ├─ All Todos
│   ├─ Today Todos
│   ├─ High Priority Todos

    ├─ Account Settings

    ├─ Display Preferences
    ├─ About Section
    └─ Data Management
```

## Responsive Design

```
All screens adapt to:
├─ iPhone SE (5.4")     - Compact
├─ iPhone 14/15 (6.1")  - Regular
├─ iPhone 14+ (6.7")    - Extra Large
├─ iPad (Portrait)      - Tablet
└─ iPad (Landscape)     - Tablet Landscape

Breakpoints:
├─ < 400px: Compact layout
├─ 400-600px: Regular layout
├─ > 600px: Expanded layout with sidebars
```

## Animation & Transitions

```
Smooth Transitions:
├─ Sheet presentation: Slide up
├─ Tab switching: Fade
├─ Score update: Animate number
├─ Completion toggle: Scale + checkmark animation
├─ Streak change: Number pop animation
└─ Habit creation: Slide in

Duration: 300-500ms (standard)
Timing: ease-in-out (smooth)
```

---

This visual guide helps understand the app's structure and layout. All screens are built with SwiftUI and support dark mode with liquid glass design elements.
